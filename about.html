About Rufus

profile

I'm an IT technologist working at the intersection of networking, cloud infrastructure, and web development — three disciplines that increasingly depend on one another.

My foundation is hands-on network operations: configuring routing devices, troubleshooting hardware, and keeping systems running in live NOC environments. That grounding shapes how I think about cloud infrastructure — every deployment is still, underneath, a network problem.

More recently I've been extending that into web development — learning JavaScript, Node.js, and Express to build the applications that sit on top of the infrastructure I already know how to run.
